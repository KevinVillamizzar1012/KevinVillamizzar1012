## 🚀 Proyecto Destacado

### 💊 [Farmacias Curarte - Sistema de Gestión Full Stack](https://github.com/KevinVillamizzar1012/farmacia-curarte)

Sistema completo para la gestión de inventarios, ventas y administración de farmacias con tres roles diferenciados.

- **Backend:** Spring Boot 3, Spring Security, JWT, JPA, MySQL.
- **Frontend:** Angular 18, TypeScript, Chart.js, Bootstrap.
- **Características:** Punto de Venta (POS), Reportes exportables (Excel/PDF), Dashboard con métricas, Subida de imágenes y Carrusel de banners.
